# Si Educational - Static HTML, CSS & JavaScript Landing Page

This is a static version of the "Si Educational" landing page, converted from the original Next.js + React + TypeScript project. It uses only pure HTML, CSS, and Vanilla JavaScript, with no build tools or frameworks.

#### Preview

 - [Original Next.js Demo](https://themewagon.github.io/si-education/)

#### Download
 - [Original Project from ThemeWagon](https://themewagon.com/themes/si-education/)

## Getting Started

To view the website, simply open the `index.html` file in your web browser. There is no need for any installation or build steps.
